Data Obtaining

A function for downloading and unpacking ZIP file, returns a name of txt file for the further processing.
In this example we used Enlgish - French sentence pairs from Tatoeba Project
Run the function and get the link to the downloaded file: 

path = 'data/' + get_data('http://www.manythings.org/anki/fra-eng.zip')

Data Cleaning

Sentense pairs need some cleaning - remove punctuation, numbers and nonprintable characters and also change to lower case.
Then wrap it into another function that works with the txt file, removes duplicates and empty entries.

Run the function and get a numpy array that can be used by Keras:

clean_data = clean_data(path)

Split Dataset in Train and Test

The whole dataset is about 100 000 rows, but for our purposes we use only 10 000. Note that dataset is sorted - phrases length increases by the end. We split our data: 80% train and 20% test.
The output is two numpy arrays, also these sets are saved on the disk with joblib for the future model evaluation.
